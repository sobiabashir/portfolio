Projects

Project # 1 : Digital Recipe Book

The task is to develop a digital Recipe Book application for households and restaurants with the ability to add, edit, display, and remove recipes. Each recipe includes a name, instructions, and a list of ingredients. The application will use an array of Recipe objects and involve multiple forms with established communication between them.

Project#2: Customer Registry

This assignment introduces fundamental concepts of object-oriented programming (OOP) by focusing on encapsulation and data-hiding. Students will learn to implement constructors, manage the "has a" relationship between objects, and utilize properties effectively. The task involves creating a customer registry application where users can add, edit, and delete customer information. Grades are based on the functionality achieved: C for basic addition of customers, and higher grades (B or A) based on code quality, structure, and documentation.

Project#3: To Do reminder

This assignment focuses on advanced C# programming concepts including enums, constructors, properties, and List. Students will also learn to integrate essential controls like MenuStrip, ToolTip, Timer, and MessageBox into a graphical user interface. The goal is to develop a ToDo Reminder application that allows users to add tasks with specific dates and times, display them in a list format on the GUI, and utilize a priority enum for task classification. The application must include functionality to save task data to files, emphasizing well-structured code, comprehensive documentation, and proper implementation of object-oriented principles.

Project#4: Multimedia Store

The MultimediaStore project showcases a robust software application designed for managing multimedia items such as movies, music albums, and books. Key features of the application include: Item Management: Users can add new items, edit existing ones, and remove items from the store's inventory.Collection Browsing: Intuitive navigation allows users to browse through collections of multimedia items efficiently.Rental or Purchase Functionality: Depending on project scope, the application may support features for renting or purchasing multimedia items.User Interface: Designed with a user-friendly interface using appropriate controls for seamless interaction and usability.
